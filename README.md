# ASCII Image Generator

A web-based tool that converts images into ASCII art. Upload an image, drag to select regions, and watch them transform into ASCII characters in real-time.

## Features

- **Drag & Select Conversion**: Click and drag over any region of your uploaded image to convert it to ASCII art
- **Multiple Character Sets**: Choose from 3 preset character sets or create your own custom set
  - Standard: `@%#*+=-:. `
  - Detailed: 70+ characters for high-fidelity gradients
  - Blocks: `█▓▒░ ` for a retro terminal look
- **Image Adjustments**: Fine-tune your ASCII output with:
  - Brightness
  - Contrast
  - Sharpness
- **Edge Detection**: Toggle Sobel edge detection with adjustable threshold
- **Invert Mode**: Flip dark and light values
- **Segment Tracking**: Each selection is logged with coordinates and customizable notes
- **Export**: Export all your ASCII segments with metadata
